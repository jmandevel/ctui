# ctui sandbox

In this sandbox example:

* if you launch app in a terminal, a second screen will take over the terminal. click the terminal to control that instead.
    * use arrow keys to move the character around the terminal
* click the GLFW window to control it.
  * use arrow keys to move the smiley face around the window
  * you can hold right click to pan and scroll to zoom
  * press r to reset the view and resize the tile width and height based on the viewport dimensions
  * press f to toggle fullscreen
  * you can close the GLFW window by pressing the x button, and the terminal will continue
* some input commands work regardless if the terminal or GLFW window are selected
  * press esc to stop the application
